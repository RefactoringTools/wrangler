Description

Remove Unreferenced Assignments - Removes assignments when the variables are not used afterwards.

For example, the expression:
begin
A = 10,
B = 20,
C = 30,
E = A + B + C,
A
end.
can be simplified by these transformations to:
begin
A = 10,
A
end.
